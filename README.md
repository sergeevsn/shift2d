# Shift2D

Desktop-приложение для применения статических поправок из csv-таблицы к 2D профилю из SEG-Y, с визуальным контролем и возможностью загрузки горизонта из таблицы.

## Возможности

- загрузка SEG-Y с выбором байтов координат трасс и превью геометрии профиля;
- импорт статики и горизонтов из CSV (по координатам X/Y или по номерам трасс);
- режимы обработки: просмотр, прямое и обратное применение statics;
- интерполяция пропусков, экстраполяция по краям, сглаживание кривой statics;
- интерактивный разрез: zoom, pan, палитры амплитуд, клиппинг, окно по трассам и времени;
- отображение кривой statics и горизонта на разрезе;
- экспорт скорректированного SEG-Y.

## Зависимости

- **CMake** 3.18 или новее
- **C++17** компилятор (GCC, Clang, MSVC, MinGW)
- **Qt 5** — модули `Core` и `Widgets` (рекомендуется 5.12+)
- **Git** — для автоматической загрузки [segyio](https://github.com/equinor/segyio) через CMake FetchContent (если локальная копия `segyio/` отсутствует)

## Сборка в Linux

1. Установите инструменты сборки и Qt5 для разработки (пакеты вроде `cmake`, `g++`, `git`, `qtbase5-dev` — названия зависят от дистрибутива).

2. Клонируйте репозиторий и перейдите в каталог проекта.

3. Создайте отдельный каталог сборки рядом с исходниками:

   ```bash
   mkdir build
   cd build
   ```

4. Настройте и соберите проект:

   ```bash
   cmake ..
   cmake --build .
   ```

   Если Qt не находится автоматически, укажите префикс Qt при конфигурации, например:

   ```bash
   cmake .. -DCMAKE_PREFIX_PATH=<каталог-установки-Qt>
   ```

5. Запуск:

   ```bash
   ./shift2d
   ```

## Сборка в Windows

1. Установите **CMake**, **Git** и **Qt 5** (MinGW или MSVC — один комплект на весь проект).

2. Клонируйте репозиторий и откройте терминал Qt или «x64 Native Tools Command Prompt» (для MSVC).

3. Создайте каталог сборки и перейдите в него:

   ```bat
   mkdir build
   cd build
   ```

4. Сгенерируйте файлы сборки и соберите проект.

   **MinGW (пример):**

   ```bat
   cmake .. -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH=<каталог-установки-Qt>
   cmake --build .
   ```

   **MSVC (пример):**

   ```bat
   cmake .. -DCMAKE_PREFIX_PATH=<каталог-установки-Qt>
   cmake --build . --config Release
   ```

   Альтернатива — открыть `CMakeLists.txt` в **Qt Creator**, выбрать kit с Qt 5 и собрать проект из IDE.

5. Исполняемый файл `shift2d.exe` появится в каталоге сборки (для MSVC — часто в подкаталоге `Release`).

   Для запуска вне Qt Creator может понадобиться, чтобы каталог с DLL Qt был в `PATH`, либо развернуть Qt-библиотеки через `windeployqt`.
